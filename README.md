# react-native-pinch-viewer

A highly customizable React Native pinch-to-zoom viewer built with gesture-handler and reanimated, providing smooth zooming, panning, and high-performance animations for images and other visual content.

## Installation


```sh
npm install react-native-pinch-viewer
```


## Usage


```js
import { multiply } from 'react-native-pinch-viewer';

// ...

const result = await multiply(3, 7);
```
